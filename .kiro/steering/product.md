# Product Overview

TriviaNFT is a blockchain-based trivia gaming platform built on Cardano that rewards players with NFTs for perfect gameplay.

## Core Features

- **Trivia Sessions**: Players answer 10 questions per session across 9 categories
- **NFT Rewards**: Perfect scores (10/10) earn mint eligibility for category-specific NFTs
- **NFT Forging**: Collect multiple NFTs to forge higher-tier Ultimate NFTs (Category, Master, Seasonal)
- **Seasonal Competition**: Leaderboards with seasonal prizes and point-based rankings
- **Daily Limits**: 10 sessions per day with cooldown periods between plays

## User Flow

1. Connect Cardano wallet for authentication
2. Select trivia category and start session
3. Answer 10 questions (10 seconds each)
4. Perfect score grants 1-hour mint eligibility
5. Mint NFT or collect for forging Ultimate NFTs
6. Compete on leaderboards for seasonal prizes

## Platform Components

- **Web PWA**: Expo-based progressive web app
- **Mobile App**: Native iOS/Android via Expo
- **API**: Serverless Lambda functions on AWS
- **Blockchain**: Cardano for NFT minting and burning
